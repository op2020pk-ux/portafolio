## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3 (Diseño moderno con estilo Cyberpunk/Neón), JavaScript Estructurado y Asíncrono (Fetch API).
* **Backend:** PHP (Estructura modular orientada a servicios API limpia).
* **Base de Datos:** MySQL conectado mediante persistencia de datos estable (XAMPP).

## ⚙️ Características Principales

* **Control de Acceso Seguro:** Panel administrativo protegido mediante autenticación por firma digital.
* **Gestión Modular (CRUD):** Capacidad para publicar, actualizar y eliminar sistemas del catálogo en tiempo real.
* **Mapeo Automatizado de Transacciones:** Registro instantáneo de solicitudes de clientes, vinculando automáticamente tokens de órdenes únicas y capturas visuales de los productos seleccionados.
* **Arquitectura Desacoplada:** Interfaz de usuario (.html) completamente independiente de la lógica del servidor (.php), comunicadas eficientemente en segundo plano.
